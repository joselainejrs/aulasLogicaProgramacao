###  Constante e Variáveis:

```
- Valor constante ela se mantem inalterada em toda execusão do algoritimo;
--- Tipos:
         
                         
- Valor variável ela muda dependendo da situação  => São areas de memoria que se 
  armazenam valores que podem ser modificados durante a execução do algoritimo;
--- Tipos:
          Númericas => Inteiro (não tem casa décimas)
                    => Real (tem casa décimas)
          Alfanuméricas => Caracteres: (letras, númeres e caracteres especias )
          Lógicos   => Verdadeiro
                    => Falso
          Operadores Aritimétricos => +   Adição
                                      -   Menos ou subtração
                                      *   Multiplicação ou Asterisco
                                      /   Divisão
                                      n/o Resto
          Operadores Relacionais  =>  São operadores que estabelecem uma comparação entre duas variáveis ou entre uma variável a uma constante. 
          O valor dessa comparação é o resultado lógico.
                  os operadores são:
                       = igual;
                       <> diferente;
                       > maior que;
                       < menor que;
                       >= maior ou igual que;
                       <= menor ou igual que;
                       concatenar (colocar junto ou juntar )

que representa diferente.
                    
          Explicações de alguns pontos:          
                todos os números em reais ele é guardado antes e depois da vírgula;
                Não conseguimos fazer conta com valor caracteres
```
